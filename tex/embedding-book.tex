% Abstract
\chapter*{Abstract}
\addcontentsline{toc}{chapter}{Abstract}

This book documents the embedding and retrieval stack built around local \texttt{llama-server}
inference. It covers the foundations of how embeddings work, how to evaluate and select
embedding models, and how to assemble a complete Retrieval-Augmented Generation (RAG) pipeline
running on consumer hardware.

The primary target is a local-first deployment: models served via \texttt{llama-server} from
GGUF artifacts, embeddings exposed through an OpenAI-compatible HTTP endpoint, and a retrieval
backend that runs without external infrastructure. The Qwen3 Embedding family is treated as the
leading local candidate throughout.

The book is organized into five parts:

\begin{enumerate}
    \item \textbf{Foundations} --- what embeddings are, and how tokenization shapes what models can process
    \item \textbf{Benchmarks and Model Selection} --- reading MTEB and RTEB scores, selecting for local RAG
    \item \textbf{Retrieval Architecture} --- dense, sparse, and ColBERT retrieval; ANN indexes; BM25 and RRF
    \item \textbf{Building the Pipeline} --- chunking strategies, the full RAG pipeline, and storage backend comparison
    \item \textbf{Local Deployment} --- serving with llama-server, query instructions, and operational guidance
\end{enumerate}
